## Markdown编辑器

笔者经常用markdown写文档，网上markdown各种编辑器很多，但感觉很不好用，甚至还有收费 。而有些在线的编辑器还不错但毕竟不如客户端常用，于是有了MDNotePad，目的就是简单实用快速(的记事本)

大体功能如下：（实际差不多就是个记事本）
- 自动识别编码打开文件
- 切换文件编码
- 实时markdown预览编辑
- 导出成html文件
- 拖拽打开文件
- 操作简单，无插件广告
